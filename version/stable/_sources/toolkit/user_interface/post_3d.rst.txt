.. _post_3d:

3D Postprocessing
=================

The menu only displays the **3D Postprocessing** panel if the design is solved and results are extracted by the toolkit.

You have five 3D postprocessing categories:

* :ref:`RCS <rcs_3d>`
* :ref:`Range profile <range_profile_3d>`
* :ref:`Waterfall <waterfall_3d>`
* :ref:`ISAR 2D <isar_2d_3d>`
* :ref:`ISAR 3D <isar_3d_3d>`

.. note::

    - You can access :ref:`3D settings <settings_3d>` to change the plot settings.
    - In the following images, **Solution** is set to **rcs_scenario**, which is for a car.

.. _rcs_3d:

RCS
===

#. Go to the **3D Postprocessing** panel from the left sidebar. In the following image, **RCS** is the selected postprocessing category.

   .. image:: ../../_static/rcs_3d.png
     :width: 800
     :alt: RCS postprocessing category
     :align: center

#. Click **Plot** to generate a 3D plot based on the specified settings.

   .. image:: ../../_static/rcs_3d_car.png
     :width: 800
     :alt: RCS plot
     :align: center
